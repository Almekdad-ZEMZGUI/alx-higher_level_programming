The ``0-add_integer`` module
======================

Using ``add_integer``
-------------------

First import ``add_integer`` from the ``0-add_integer`` module:

    >>> from 0-add_integer import add_integer

Normal test cases:

	>>> add_integer(1, 2)
	3
	>>> add_integer(-1, -2)
	-3
	>>> add_integer(100, -2)
	98
	>>> add_integer(-100, 2)
	-98
	>>> add_integer(100.3, 2.5)
	102
